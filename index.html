IEO_Group36

This website is a portfolio of all the important work and assignments we have been doing during the last 11 weeks of our studies in 1 semester at fontys.

Terminal or command line

Using the windows command line and linux terminal, we did some action such as creating, coping, moving and deleting files and directories. listing all the files and directories that are inside a specific directory and more other commands.

Git Repository

For our group projects,we used Git lab which is a distributed version-control system for tracking changes in source code during software development. It is designed for coordinating work among programmers, but it can be used to track changes in any set of files.


Network Basics

Netkit commands have been used to start and halt network nodes. We generated HTTP traffic, captured the network traffic. Looking at your captured packets, we found details such as the source and destination MAC address of this HTTP packet, the source and destination IP address of this HTTP packet; the source and destination port of this HTTP packet; and the host name of this HTTP Get packet.

IP Routings

Our task was to add routing information to the nodes in such a way, that every node of your network should be able to ping any other node of your network. The routes were be optimal, so the shortest path from node to node should be used.


TCP-UDP

TCP and UDP are two most used transport layer protocols. TCP and UDP provide peer-to-peer communication for the application processes. Using Wireshark, we thought about different scenarios for TCP use and UDP. We simulated the scenarios by opening the Wireshark trace showing UDP and TCP packets.
